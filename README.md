# init-yiyousport-journal

## 项目简介

`init-yiyousport-journal` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，主要用于存放和展示各类静态页面资源，便于统一管理和访问。

## 目录说明

```
.
├── index.html          # 仓库首页 / 导航页
├── pages/              # 存放独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 静态资源目录
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有归档的独立 HTML 页面。
- `assets/`：页面引用的公共资源（样式、脚本、图片等）。
- `index.html`：可选的入口页面，用于列举或导航到各个归档页面。

## 页面归档说明

本仓库中的每个 HTML 页面均为独立文件，互不依赖。  
页面可能包含简单的样式与脚本，但原则上不涉及后端服务或外部 API 调用。

归档页面主要用于以下场景：
- 信息展示
- 工具页面
- 示例页面
- 临时或实验性页面

如需添加新页面，请遵循以下建议：
- 将 HTML 文件放入 `pages/` 目录
- 保持文件名简洁且具有描述性
- 如需引用公共资源，请使用相对路径指向 `assets/` 目录

## 维护说明

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署本仓库内容。
